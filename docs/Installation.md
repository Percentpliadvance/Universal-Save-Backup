# Installation Guide

## Requirements

* Windows 10 or Windows 11
* .NET Runtime 8.0 or newer

## Installation

1. Download the latest release from the Releases page.
2. Extract the ZIP archive.
3. Open the extracted folder.
4. Run UniversalSaveBackup.exe.
5. Follow the setup instructions.

## First Backup

1. Open Universal Save Backup.
2. Click "Add Game".
3. Select the save folder.
4. Click "Create Backup".
5. Verify that the backup appears in Backup Manager.

## Updating

1. Download the latest release.
2. Replace existing application files.
3. Launch the updated version.

## Troubleshooting

If a backup cannot be created:

* Verify folder permissions.
* Ensure enough free disk space.
* Check that the save directory exists.
