# Frequently Asked Questions

## What is Universal Save Backup?

Universal Save Backup is a desktop utility that helps create and restore backups of game save files.

## Does the application modify game files?

No. The application only creates copies of save data.

## Is internet access required?

No.

## Can I create multiple backups?

Yes. Multiple backups can be stored for each game profile.

## Can I restore older backups?

Yes. Previous backups remain available in Backup Manager.

## Is my data uploaded anywhere?

No. All backups are stored locally on your device.

## Does the application support Steam games?

Game save folders can be added manually regardless of platform.

## How do I remove a game profile?

Open the profile manager and remove the selected entry.
