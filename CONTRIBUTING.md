# Contributing

Thank you for your interest in Universal Save Backup.

## How to Contribute

You can help by:

- Reporting bugs
- Suggesting features
- Improving documentation
- Testing releases
- Submitting pull requests

## Development Setup

1. Fork the repository.
2. Clone your fork.
3. Create a new branch.
4. Make your changes.
5. Open a pull request.

## Branch Naming

Use clear branch names:

```text
feature/scheduled-backups
fix/restore-path-check
docs/update-installation
